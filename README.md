# 🇮🇳 BharatVoice AI
### Apni Bhasha Mein, Apna Haq — Your Rights, In Your Language

## 🎯 Problem Statement
Every year, thousands of crores worth of government welfare 
benefits go unclaimed in India — not because people are 
ineligible, but because the information is locked behind 
complex English documents that ordinary citizens cannot 
understand.

## 💡 Solution
BharatVoice AI is a multilingual AI agent that transforms 
any government scheme, scholarship, or policy into a simple 
voice conversation in the citizen's own language.

A farmer in Maharashtra, a student in Bihar, or a shopkeeper 
in Tamil Nadu can simply speak in their mother tongue and 
instantly receive clear guidance on eligibility, benefits, 
application process, and required documents.

## ✨ Features
- 🤖 AI-powered chat using Google Gemini
- 🎤 Voice input using Web Speech API
- 🔊 Text-to-speech for all AI responses
- 🌐 5 languages — English, Hindi, Marathi, Tamil, Telugu
- ✅ Smart eligibility checker (6-question quiz)
- 📄 Auto-generated application letters
- 🏛️ 20+ real Government of India schemes
- 📊 Impact dashboard with live charts

## 🌐 Adaption Track Integration
Built on Adaption's 242-language AI data platform 
(adaptionlabs.ai). Our government schemes dataset was 
ingested, diagnosed, translated and exported directly 
through the Adaption platform.

Dataset open-sourced on:

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript (Single file)
- AI: Google Gemini API (gemini-1.5-flash)
- Voice: Web Speech API
- Data Platform: Adaption (adaptionlabs.ai)
- Charts: Chart.js

## 🚀 How to Run
1. Clone this repository
2. Open index.html in Chrome
3. Add your Gemini API key on line 441
4. That's it — no installation needed!
